# \# Dataset Access

# 

# This project uses the CIC IDS 2018 dataset for intrusion detection experiments.

# 

# The dataset is publicly available and can be obtained from the official source:

# https://www.kaggle.com/datasets/solarmainframe/ids-intrusion-csv 

# 

# Due to licensing restrictions, the dataset is not included in this repository.

# 

# \## Usage

# 

# The experiments use flow level features derived from the dataset. Feature normalization and selection follow the procedure described in the paper. No additional preprocessing steps beyond those reported are required.
